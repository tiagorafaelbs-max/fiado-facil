FiadoApp

Política de Privacidade

Última atualização: 22 de junho de 2026

Resumo simples: O FiadoApp coleta apenas os dados necessários para que o app funcione. Não vendemos suas informações para terceiros. Seus dados de clientes e vendas são seus.

1. Quem somos

O FiadoApp é um aplicativo de controle de vendas a prazo (fiado) desenvolvido para pequenos comerciantes brasileiros. Responsável: Tiago Rafael, acessível pelo e-mail tiagorafael.bs@gmail.com.

2. Quais dados coletamos

Coletamos apenas o necessário para o funcionamento do app:

  • Dados de cadastro: nome do negócio e endereço de e-mail para criação de conta.
  • Dados de clientes: nome, telefone, CPF, empresa e endereço — informados por você ao cadastrar seus clientes.
  • Dados de transações: valores de vendas, pagamentos, datas e descrições que você registra.
  • Dados de uso: informações técnicas como versão do app, tipo de dispositivo e erros — usados para melhorar o app.
  • Notificações: se você permitir, coletamos o token do dispositivo para enviar lembretes de vencimento.

3. Como usamos seus dados

  • Fornecer e manter o funcionamento do app.
  • Enviar notificações de vencimento de cobranças (somente se você autorizar).
  • Gerar relatórios e análises dentro do próprio app para você.
  • Melhorar a experiência e corrigir erros técnicos.
  • Processar pagamentos de assinatura via MercadoPago.

4. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros para fins comerciais. Os dados podem ser compartilhados somente com:

  • Supabase: nosso provedor de banco de dados e autenticação, com servidores nos EUA, em conformidade com as leis de proteção de dados.
  • MercadoPago: para processar pagamentos de assinatura, conforme a política de privacidade deles.
  • Autoridades legais: quando exigido por lei ou ordem judicial.

5. Armazenamento e segurança

Seus dados são armazenados de forma segura na plataforma Supabase, com:

  • Criptografia em trânsito (HTTPS/TLS).
  • Isolamento por usuário via Row Level Security — nenhum usuário acessa dados de outro.
  • Backups automáticos diários.

6. Seus direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Acesso: solicitar uma cópia de todos os seus dados.
  • Correção: corrigir dados incompletos ou desatualizados.
  • Exclusão: solicitar a exclusão da sua conta e todos os dados associados.
  • Portabilidade: receber seus dados em formato estruturado.
  • Revogação: revogar consentimentos dados anteriormente.

Para exercer qualquer um desses direitos, entre em contato pelo WhatsApp +55 (31) 99551-5045 ou e-mail tiagorafael.bs@gmail.com.

7. Exclusão de conta

Você pode solicitar a exclusão completa da sua conta e todos os seus dados a qualquer momento entrando em contato conosco. Os dados são removidos permanentemente em até 30 dias após a solicitação.

8. Cookies e rastreamento

O aplicativo mobile não utiliza cookies. A versão web pode utilizar armazenamento local (localStorage) apenas para manter sua sessão ativa, sem rastreamento de terceiros.

9. Menores de idade

O FiadoApp é destinado a maiores de 18 anos. Não coletamos dados de menores de idade de forma intencional.

10. Alterações nesta política

Podemos atualizar esta política periodicamente. Quando isso ocorrer, notificaremos você pelo e-mail cadastrado ou por aviso dentro do app. O uso contínuo do app após as alterações implica aceite da nova política.
